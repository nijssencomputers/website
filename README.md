# Nijssen Computers Website 2026

## Upload naar webhost

Upload de volgende bestanden naar de root van je webhost via FTP:

- index.html (hoofdpagina)
- style.css (stylesheet)
- robots.txt (voor zoekmachines)
- _files/ (eventuele extra bestanden)

## FTP instructies

1. Verbind met je webhost via FTP (gebruik bijvoorbeeld FileZilla)
2. Upload alle bestanden naar de public_html of www directory
3. Controleer of index.html als startpagina wordt herkend
